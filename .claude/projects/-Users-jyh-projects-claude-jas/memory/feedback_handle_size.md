---
name: Handle size preference
description: User prefers larger control handles (10px draw, 8px hit radius) over the default 6px — easier to grab with the mouse.
type: feedback
---

Control handles at 6px were too small to find with the mouse. Increased to 10px draw size and 8px hit radius across all three languages. User confirmed this feels good.

**Why:** Small handles are hard to click, especially on Retina displays.
**How to apply:** When adding new interactive handles or control points, use 10px draw size and 8px hit radius as the baseline.
